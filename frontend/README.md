# 収支管理フロントエンド

Vue 3 + TypeScript + Vite で作成した収支管理のモバイル向けUIです。

## APIの設定

APIのベースURLは `src/config.ts` で変更できます。デフォルトは `http://127.0.0.1:8000` です。

```ts
export const API_BASE_URL = 'http://127.0.0.1:8000'
```

## 開発

```bash
npm install
npm run dev
```

## ビルド

```bash
npm run build
```

## 機能

- **一覧**: 今日から過去10日間の利用日の支出を表示（利用日・目的・金額）。表ヘッダは固定で本体のみスクロール。
- **新規入力**: 右下の「+」ボタンから入力画面を開く。利用日（カレンダー）、目的、金額、支出元、支払日（支出元選択で自動算出・手動変更可）。予算名は「未分類」固定。
- **編集**: 一覧の行をタップして同じフォームで変更可能。
